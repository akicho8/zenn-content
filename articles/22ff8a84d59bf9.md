## END {} で終了処理の登録

```ruby
END { p 3 }
END { p 2 }
p 1
# >> 1
# >> 2
# >> 3
```

ただ、複数行になるから do end ブロックで書こうかなってするとエラーになる

```ruby
END do
end
# ~> -:1: syntax error, unexpected `do', expecting '{'
# ~> END do
# ~>     ^~
```

じゃあ proc にしとくか

```ruby
END(proc{})
# ~> -:1: syntax error, unexpected '(', expecting '{'
# ~> END(proc{})
# ~>    ^
# ~> -:1: syntax error, unexpected ')', expecting end-of-input
# ~> END(proc{})
# ~>           ^
```

え？
素直に `at_exit {}` を使った方が良さそうだ。
